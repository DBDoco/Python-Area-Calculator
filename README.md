## How To Use

Clone this repository
```bash
$ git clone https://github.com/DBDoco/Python-Area-Calculator.git
```

Go into the repository
```bash
$ cd Python-Area-Calculator
```
Install packages
```bash
$ pip install scipy numpy matplotlib sympy
```

Run the script
```bash
$ py Calculator.py
```
